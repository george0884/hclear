# hclear
SCRIPT para instalar paquetes , actualizar o eliminar  ppa solo para Ubuntu o Derivados



Permisos :
```
sudo chmod +x hclear.sh
```

Ejecutar:

```
./hclear.sh

```
Solo debes escribir un numero
